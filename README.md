# Čišćenje Dugo Selo

Lokalna web stranica za uslugu čišćenja stanova i kuća u Dugom Selu i okolici.

## Tehnologija

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS 4

## Pokretanje

```bash
npm install
npm run dev
```

Stranica je dostupna na [http://localhost:3000](http://localhost:3000).

## Kontakt podaci

Ažurirajte telefon, WhatsApp i email u `src/lib/constants.ts`.

## Stranice

- `/` — Početna
- `/ciscenje-stanova-dugo-selo` — SEO landing za čišćenje stanova
- `/generalno-ciscenje-stana-dugo-selo` — SEO landing za generalno čišćenje
- `/ciscenje-kuca-dugo-selo` — SEO landing za čišćenje kuća
- `/pranje-prozora-dugo-selo` — SEO landing za pranje prozora
